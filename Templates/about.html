About StudentAI

An Artificial Intelligence powered web application that predicts whether a student is likely to pass or fail based on academic and personal factors.

Machine Learning Model

Algorithm: Random Forest Classifier

Accuracy: 92.41%

Framework: Scikit-learn Pipeline

Language: Python

Dataset

Student Performance Dataset

395 student records

Academic performance dataset from Kaggle

Binary prediction: Pass or Fail

Input Features

  • Gender
  • Age
  • Study Time
  • Previous Failures
  • Absences
  • First Grade (G1)
  • Second Grade (G2)
  • Higher Education Interest

Technology Stack

  • Django
  • Python
  • Scikit-learn
  • Pandas
  • Joblib
  • Bootstrap 5
  • HTML5 & CSS3
  • SQLite

How the AI Works

1. User Input

Student information is entered through the prediction form.

2. Preprocessing

The Scikit-learn pipeline automatically prepares the data.

3. AI Prediction

The Random Forest model predicts whether the student is likely to pass or fail.

4. Results

The system displays the prediction, confidence score and recommendation.
